Use the generator to generate custom component templates with the `bit create` command. All available components can be seen when running `bit templates`.

### Configuring the Generator

Add the following to your `workspace.jsonc` file to enable the generator:

```json
"teambit.generator/generator": {
  "aspects": ["company.scope/generator/component-templates"]
}
"teambit.base-react/generator/component-templates": {}
```

This registers the component aspect so that your templates will appear in the CLI when you run `bit templates`.

```bash
bit templates
```

### Using the Generator

Use your generator to create the component files

```bash
bit create my-react ui/button
```
