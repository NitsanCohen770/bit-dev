import { Button } from './button';

### Component Usage

Add the prop of secondary to get a secondary button. Accepts children so that text, icons etc can be rendered inside.

```js
<Button>Get Started</Button>
```

### Live Playground

Modify the props to see it change live.

```js live
<Button secondary>Watch video</Button>
```
