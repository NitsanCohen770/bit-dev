## Overview

Shows the benefits section.

### Component usage

Import the mock data from the entity component and pass it to the benefits prop to show the list of benefits.

```js
import { benefits } from '@teambit/community.entity.benefits';

<BenefitsSection benefits={benefits} title="Build better. Build scalable. " />;
```
