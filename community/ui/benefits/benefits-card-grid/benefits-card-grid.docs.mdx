## Overview

A grid of cards showing the benefits.

### Component usage

Import some mock data from an external source and pass it to the `benefits` prop.

```js
import { benefits } from '@teambit/community.entity.benefits';

<BenefitsCardGrid benefits={benefits} />;
```

### Live Playgournd

Add an array of use cases to the `benefits` prop.

```js live
<BenefitsCardGrid
  benefits={[
    {
      title: 'Distributed',
      text: 'Component ownership is distributed across developer teams',
      image: 'brands/logo-react.svg',
      alt: 'react logo'
    },
    {
      title: 'Maintainable',
      text: 'Components are easy to replace and refactor over time.',
      image: 'brands/logo-angular.svg',
      alt: 'angular logo'
    }
  ]}
/>
```
