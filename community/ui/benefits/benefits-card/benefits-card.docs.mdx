## Overview

Shows the product's benefits with an image, title and text.

### Component usage

```js
<BenefitsCard
  title="Distributed"
  text="Component ownership is distributed across developer teams"
  image="brands/logo-react.svg"
  alt="React logo"
/>
```

### Live Playground

Modify the props to see it change live:
