Sticky menu that stays on the screen as you scroll down.

### Component Usage

Add an array of links to the `links` prop.

```js
<StickyMenu
  links={[
    {
      href: '#compose',
      text: 'Compose'
    },
    {
      href: '#Collaborate',
      text: 'Collaborate'
    }
  ]}
/>
```
