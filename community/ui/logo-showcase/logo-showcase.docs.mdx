## Overview

A strip of images with gentle animation. Lazy loading is added by default for all images but can be overridden with the `loading` prop.

### Component usage

Import some mock data from an external source and pass it to the `images` prop.

```js
import { clientLogos } from '@teambit/community.entity.images';

<LogoShowcase data-testid="test-logo-showcase" logos={clientLogos} />;
```

### Live Playgournd

Add an array of images to the `images` prop.

```js live
<LogoShowcase
  images={[
    {
      src: 'https://static.bit.dev/homepage-enterprise-logos/Dell.svg',
      alt: 'dell'
    },
    {
      src: 'https://static.bit.dev/homepage-enterprise-logos/Carlsberg-1.svg',
      alt: 'Carlsberg'
    }
  ]}
/>
```
