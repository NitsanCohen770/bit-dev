## React Component for rendering text

A basic div that renders some text

### Component usage

```js
<UseCaseCard
  title="Component Library"
  text="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Duis semper, quam sed."
  icon="Ripple-pending"
></UseCaseCard>
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<UseCaseCard
  title="Component Library"
  text="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Duis semper, quam sed."
  icon="Ripple-pending"
></UseCaseCard>
```
