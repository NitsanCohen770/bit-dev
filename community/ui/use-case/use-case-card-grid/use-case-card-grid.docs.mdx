## Overview

A grid of cards showing the use cases.

### Component usage

Import some mock data from an external source and pass it to the `useCases` prop.

```js
import { useCases } from '@teambit/community.entity.use-cases';

<UseCaseCardGrid useCases={useCases} />;
```

### Live Playgournd

Add an array of use cases to the `useCases` prop.

```js live
<UseCaseCardGrid
  useCases={[
    {
      title: 'Component Library',
      text: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Duis semper, quam sed.',
      image: 'brands/logo-react.svg',
      alt: 'React logo',
      href: '/react'
    },
    {
      title: 'Component Library',
      text: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Duis semper, quam sed.',
      image: 'brands/logo-react.svg',
      alt: 'React logo',
      href: '/react'
    }
  ]}
/>
```
