## React Component for rendering text

A basic div that renders some text

### Component usage

```js
import { useCases } from '@teambit/community.entity.use-cases';

<UseCaseSection
  useCases={useCases}
  title="These are no longer a challenge"
  href="/"
  linkText="See more use cases"
/>;
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<UseCaseSection
  useCases={useCases}
  title="These are no longer a challenge"
  href="/"
  linkText="See more use cases"
/>
```
