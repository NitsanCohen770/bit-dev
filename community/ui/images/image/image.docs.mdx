import { Image } from './image';

## Concrete image, using the Static Storage CDN

Treats src as a relative path on top of the image storage url, and supports all other properties of the html `<img>` tag.

### Component usage

```js
<Image src="homepage-bit/map.png" alt="world" fullWidth />
```
