Shows the features section.

### Component Usage

Import the mock data from the entity component and pass it to the features prop to show the list of features.

```js
import { features } from '@teambit/community.entity.features';

<FeaturesSection features={features} title="Build better. Build scalable. " />;
```

### Live Playgournd

Modify the props to see it change live:

