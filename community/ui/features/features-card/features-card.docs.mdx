## Overview

Shows the product's benefits with an image, title and text.

### Component usage

```js
<FeaturesCard
  title="Distributed"
  text="Component ownership is distributed across developer teams"
  image="https://static.bit.dev/Community/features/distributed.svg"
  alt="distribution of components"
/>
```

### Live Playground

Modify the props and see the changes in the live playground.
