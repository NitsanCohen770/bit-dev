A card component for a Use Case. Component contains:

- an image with alt tag
- a linkable title
- a text

### Use Cases Card

```js live

  title="Component Library"
  text="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Duis semper, quam sed."
  image="https://storage.googleapis.com/static.bit.dev/Community/use-cases/library.svg"
  alt="library of frameworks"
  href="/"
/>
```
