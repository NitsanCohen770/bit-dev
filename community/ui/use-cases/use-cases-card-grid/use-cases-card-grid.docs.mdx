## Overview

A grid of cards showing the use cases.

### Component usage

Import some mock data from an external source and pass it to the `useCasess` prop.

```js
import { UseCases } from '@teambit/community.entity.use-cases';

<UseCasesCardGrid useCases={useCasess} />;
```

### Live Playgournd

Add an array of use cases to the `useCasess` prop.

