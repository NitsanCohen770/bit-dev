Complete section component for the Use Cases to be used in a page component. This component contains:

- the section title
- a link to see more
- a grid of use case cards

### Section showing Use Cases with imported Json

Imports mock data in json format from the entity component which contains an array of use cases.

```js
import { useCases } from '@teambit/community.entity.use-cases';

<UseCasesSection
  useCases={useCases}
  title="These are no longer a challenge"
  href="/"
  linkText="See more use cases"
/>;
```
